##Videre

A performant local pg inspector.

The system has a tab based layout similar to vscode. A sidebar where the users can navigate between tables for the selected db. A tab system where tables are opened in tabs.

Keep the depencencies to as few as possible. This to limit supply-chain vulnerabilities. Discuss before adding if there is a potential package that we deem too valuable not to add to the project

Use zustand for state management. Tab system included.

No router package needed.
